Vid-
Monster.

Have you played retro games in Nintendo Game Boy? Do you like to play retro games like old Pokémon series?

Vid-Monster is a retro style game application, as similar as old Pokémon series. Player needs to defeat the vid-monster by choosing various actions such as 'social distancing', 'wash hands', 'mask wearing' and 'vaccine shot'. We need a hero to end this pandemic! This application is built with React and Firebase.
